## Install instructions

To install this package:

Extract this package in the root of your web site. If your site is <code>http://www.example.com</code>, you should be able to access a file named <code>http://www.example.com/favicon.ico</code>.

Insert the following code in the `head` section of your pages:

    <link rel="apple-touch-icon" sizes="180x180" href="https://edcstiftung.eu/apple-touch-icon.png">
    <link rel="icon" type="image/png" sizes="32x32" href="https://edcstiftung.eu/favicon-32x32.png">
    <link rel="icon" type="image/png" sizes="16x16" href="https://edcstiftung.eu/favicon-16x16.png">
    <link rel="manifest" href="https://edcstiftung.eu/site.webmanifest">
    <link rel="mask-icon" href="https://edcstiftung.eu/safari-pinned-tab.svg" color="#003399">
    <meta name="msapplication-TileColor" content="#2b5797">
    <meta name="theme-color" content="#ffffff">

*Optional* - Check your favicon with the [favicon checker](https://realfavicongenerator.net/favicon_checker)
